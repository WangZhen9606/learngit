这个项目为了实现一个微信机器人助手
##2019-12-01
	本来是想通过微信公众来实现对树莓派进行控制的，但是实际上手后发现可玩性还是很高的，所以就有了设计一个基于微信公众平台的机器人助手。
	目前预先设计的项目框架如下：
A.功能系统：
1.自动聊天机器人，无需验证身份；
2.用户自行设置提醒事项，如日常喝水、锻炼、纪念日等定时提醒服务；
3.小游戏模块，如成语接龙、比大小等功能；
4.智能家居模块，通过微信公众号获取家中信息，也可进行控制。
Ps：其中2-4功能需要验证用户身份，为各用户单独设置服务，因此除上述功能模块外还需要其他支持系统。
B.支持系统：
1.身份验证模块，对用户身份进行识别，对用户权限进行设置；
2.数据库相关模块，用于存储整个项目各系统各模块信息、数据等。
Ps：该架构并非最终架构，只是本人在没有任何网络编程经验基础下做出的预设，随着开发经验的提升可能随时修改。

##2019-12-3
	通过阅读微信公众开发平台相关文档，了解到个人订阅号是不能主动发送消息的，原先设计的部分功能将没有实施条件，比如提醒服务。
